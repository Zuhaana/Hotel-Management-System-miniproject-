# Hotel Management System (Python + MySQL)

## Description
A console-based Hotel Management System built using Python and MySQL.
The project supports guest login, room booking, food ordering, parking allocation,
and admin revenue reports with persistent database storage.

## Technologies Used
- Python
- MySQL
- mysql-connector-python

## Features
- Admin & Guest authentication
- Room management with availability
- Room booking & cancellation
- Food ordering per booking
- Parking allocation
- Admin revenue report

## How to Run
1. Create MySQL database `hotel_db`
2. Run SQL scripts to create tables
3. Update DB credentials in Python files
4. Run:
   ```bash
   python "Hotel Management System.py"
